# 接口说明

1. 明信片列表
```json

{
    "path" : "",      // 图片路径
    "info" : [
        {
            "createTime" : String,    // 创建时间
            "pic" : String            // 图片名
        }
    ]
}

```

2. 土特产列表
```json

{
    "path" : "",
    "info":[
        {
            "isOwn" : false     // 是否已拥有,false 否,true 是,默认false
        }
    ]
}
```
